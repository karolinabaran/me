Coding

WHY CODING?

I've started learning programming on the beginning of 2016. I really enjoyed it, and I'm getting more and more interested in it. The IT community is very open and friendly, and IT people usually like playing board games:). All that convinced me to get more into programming and change my professional path! I decided to find a job in IT. I'm still really happy and excited for that idea.


LEARNING PROGRAMMING

I was learning programming by myself on Code School, Codecademy and other tutorials. During this year I've also taken part in a few totally awesome workshops dedicated for ladies. I was on Django Girls workshop in Łódź, where we learnt the basics of Python and Django framework. Then I visited Poznań for Test Carrots workshop, where I learnt a few things about app testing. In October I took part in a PyconPL conference - the totally great Python meeting with lots of workshops, talks, interesting people and guests. Before the conference I attended PyLadies workshop, where we learnt more about Flask framework. The most important thing for my coding experience was the front-end programming bootcamp at Coders Lab programming school. It lasted 2 months, and finished in October. We had 240 hours of coding classes with great mentors, lots of homeworks and lots of learning. The course was very interesting and challenging. The classes concentrated on front-end issues: HTML, CSS and Sass, RWD, Java Script, jQuery, the basics of PHP and Wordpress. The bootcamp gave us a good basis to be a junior front-end developer.
